# 实验 01: 基线模型校准性能评估

> **实验日期**: 2026-03-31 15:26  
> **设备**: Tesla T4 (GPU 1)

## 实验配置

- 模型: MLP (Embedding + MLP)
- Embedding 维度: 16
- 隐藏层: [256, 128, 64]
- 训练轮数: 3
- 批次大小: 1024
- 学习率: 0.001
- 数据集: IVR Sample v16 CTCVR

## 训练过程

| Epoch | Train AUC | Val AUC | Val ECE | 时间 |
|-------|-----------|---------|---------|------|
| 1 | 0.7797 | 0.7932 | 0.0269 | 59.2s |
| 2 | 0.7954 | **0.7981** | 0.0065 | 57.7s |
| 3 | 0.8019 | 0.7992 | 0.0087 | 57.9s |

## 测试集结果

| 指标 | 值 | 说明 |
|------|-----|------|
| **AUC** | **0.8010** | ✅ 排序能力良好 |
| **LogLoss** | 0.5228 | |
| **ECE** | **0.0076** | ✅ 校准优秀！ |
| **MCE** | 0.0452 | |
| **PCOC** | **1.0172** | ✅ 非常接近 1.0 |
| 预测 CTR | 0.4234 (42.34%) | |
| 实际 CTR | 0.4163 (41.63%) | |

## 核心发现

### ✅ 模型已经很好校准！

1. **PCOC = 1.0172**
   - 理想值为 1.0
   - 实际值只高出 1.72%
   - 预测概率非常接近真实概率

2. **ECE = 0.0076**
   - 平均校准误差只有 0.76%
   - 非常低，说明模型预测概率可靠

3. **预测 vs 实际 CTR**
   - 预测: 42.34%
   - 实际: 41.63%
   - 差距: 0.71 个百分点

### 📊 校准性能分析

```
预测 CTR = 0.4234
实际 CTR = 0.4163
PCOC = 1.0172 → 预测略高于实际，但偏差很小
ECE = 0.0076 → 平均校准误差极低
```

## 意义

**基线模型已经具有良好的校准性能！**

这意味着：
1. 不需要额外的后处理校准（Temperature Scaling 等）
2. 模型预测的概率可以直接用于决策
3. 接下来的实验需要验证校准方法是否能进一步提升
