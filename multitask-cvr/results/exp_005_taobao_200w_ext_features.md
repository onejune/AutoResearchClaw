# exp_005：多任务 CVR 建模（Taobao，200万样本，扩展统计特征+DirectCTCVR对比）

## 实验信息

| 项目 | 内容 |
|------|------|
| 日期 | 2026-03-25 |
| 数据集 | Taobao UserBehavior |
| 样本量 | 200万（train=160万，val=20万，test=20万） |
| 正样本率 | CTR≈9.6%，CVR≈3.9% |
| 稀疏特征 | user_id, item_id, category_id |
| 数值特征 | **20个**（在 exp_003 的 12 个基础上新增 8 个） |
| Epochs | 1 |
| Batch size | 4096 |

## 新增特征（8个，在 exp_003 基础上）

| 特征 | 说明 |
|------|------|
| uc_buy_rate | 用户在该类目的 buy/pv 比率（用户-类目交叉偏好） |
| uc_cart_rate | 用户在该类目的 cart/pv 比率 |
| i_fav_cnt | 物品总 fav 次数（log1p + z-score） |
| i_fav_rate | 物品 fav/pv 比率 |
| u_fav_cnt | 用户总 fav 次数（log1p + z-score） |
| u_active_score | 用户综合活跃度 = (buy×3 + cart×2 + fav×1) / (pv+1) |
| c_pv_cnt | 类目总 pv 次数（log1p + z-score） |
| c_cart_rate | 类目 cart/pv 比率 |

## 实验结果

| 模型 | CTR AUC | CVR AUC | CTCVR AUC | 训练时间 |
|------|---------|---------|-----------|---------|
| **DirectCTCVR**（单任务基线） | — | — | 0.9511 | 89s |
| Shared-Bottom | 0.8776 | 0.9144 | 0.9373 | 93s |
| ESMM | 0.8810 | 0.9509 | 0.9484 | 94s |
| **MMoE** | **0.8900** | 0.9375 | **0.9539** | 99s |
| ESCM2 | 0.8719 | **0.9573** | 0.9483 | 92s |

## 与 exp_003（12个特征）对比

| 指标 | exp_003（12特征） | exp_005（20特征） | 提升 |
|------|-----------------|-----------------|------|
| CTR AUC（最佳） | 0.7660 | 0.8900 | **+12.4千分点** |
| CVR AUC（最佳） | 0.8969 | 0.9573 | **+6.0千分点** |
| CTCVR AUC（最佳） | 0.8800 | 0.9539 | **+7.4千分点** |

## 关键发现

1. **DirectCTCVR（单任务）CTCVR=0.9511**，与多任务模型相当——说明 20 维统计特征本身已非常强，直接建模 buy 信号就能取得很好效果
2. **MMoE** CTCVR 最高（0.9539），CTR 也最高（0.8900），多专家结构优势明显
3. **ESCM2** CVR 最高（0.9573），反事实正则化对 CVR 估计有帮助
4. 用户-类目交叉特征（uc_buy_rate/uc_cart_rate）贡献显著，CTR 提升 +12.4 千分点
