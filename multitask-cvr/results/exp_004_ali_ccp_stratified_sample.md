# exp_004：多任务 CVR 建模（Ali-CCP，分层采样，~285万样本）

## 实验信息

| 项目 | 内容 |
|------|------|
| 日期 | 2026-03-25 |
| 数据集 | Ali-CCP（天池 408） |
| 采样策略 | click=0 保留5%，click=1,buy=0 保留50%，buy=1 全保留 |
| 样本量 | train=257万，val=29万，test=290万（总计约576万） |
| 稀疏特征 | 15个（f_101~f_301） |
| 数值特征 | 0 |
| Epochs | 1 |
| Batch size | 4096 |
| 设备 | CPU |

## 采样后正样本率

- click 正样本率：约 3.2%（原始 3.89%，click=0 降采样后比例上升）
- buy 正样本率：约 0.32%（原始 0.02%，提升约 16 倍）

## 实验结果

| 模型 | CTR AUC | CVR AUC | CTCVR AUC | 训练时间 |
|------|---------|---------|-----------|---------|
| Shared-Bottom | 0.5440 | 0.5374 | 0.4914 | 389s |
| ESMM | 0.5466 | 0.5738 | 0.5336 | 389s |
| MMoE | 0.5481 | 0.5531 | 0.5147 | 400s |
| **ESCM2** | 0.5468 | **0.5790** | **0.5365** | 391s |

## 与 exp_001（Ali-CCP 原始50万样本）对比

| 指标 | exp_001（原始） | exp_004（分层采样） | 变化 |
|------|----------------|-------------------|------|
| CTR AUC（最佳） | 0.5059 | 0.5481 | +4.2千分点 |
| CVR AUC（最佳） | 0.5686 | 0.5790 | +1.0千分点 |
| CTCVR AUC（最佳） | 0.5531 | 0.5365 | -1.7千分点 |

## 结论

- 分层采样有一定帮助，CTR AUC 提升 4.2 千分点
- 但整体 AUC 仍在 0.54-0.58，远低于 Taobao 数据集（0.76-0.90）
- **根本原因**：Ali-CCP 缺乏有效的数值/统计特征，只有稀疏 ID 特征，信息量有限
- ESCM2 在 CVR/CTCVR 上表现最好，符合其反事实正则化的设计初衷
- 训练时间较长（~390s/模型），因为样本量达到 257 万

## 下一步建议

- Ali-CCP 方向：参考 Taobao 的做法，从 skeleton 中构造用户/物品统计特征
- 或直接聚焦 Taobao 数据集（特征更丰富，AUC 更高，更有研究价值）
