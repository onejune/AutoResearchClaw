## 结论

1. **H1 验证**: bt_16 AUC=0.50，完全失效；bt_10 AUC=0.65，泛化较差。OOD 问题确实存在。
2. **H2 验证**: PCOC 从 0.45 (bt_8) 到 1.10 (bt_10) 差异巨大，跨域校准性极不稳定。
3. **规律**:
   - 正样本率越高的 BT（bt_16=66%），模型越倾向低估（PCOC 偏低）
   - bt_7 AUC 最高（0.91），但 PCOC 仍偏低（0.53），说明排序能力强但绝对值校准差
4. **下一步**: Phase 1 分析哪些特征是跨域稳定的"因果特征"，哪些是虚假相关特征
