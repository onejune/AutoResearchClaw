## 🏗️ 模型架构

```
[context(5) + bid(1)] → Gradient Boosting (100 trees) → 分位数预测 [Q0.1, Q0.25, Q0.5, Q0.75, Q0.9]
```

**核心创新**:
- 使用梯度提升进行分位数回归
- 为每个分位数训练独立模型
- 提供 win probability 的完整分布
